Why is useEffect used here instead of useMemo?
✦
At 12:34, the tutor uses useEffect because this code runs a side effect (API call) — not a derived value. useMemo is for computations that return memoized values. Since fetching data triggers an external action rather than returning a computed result, useEffect is the correct hook here.

FAQ

Common Questions

What exactly is AskMyTuto? +
AskMyTuto is an AI-powered learning assistant for YouTube tutorials. It tracks your video timestamp in real time and lets you ask questions about the exact moment you're watching — giving you instant, context-aware answers without ever leaving the page.
Does it work with any YouTube video? +
Yes. Paste any public YouTube URL and AskMyTuto fetches the full transcript instantly. Support for other sources (video files, documentation) is coming soon.
Can I get answers in my own language? +
Yes. You can set your preferred output language and chat in it — even if the original tutorial is in a completely different language. AskMyTuto translates context, not just words.
What does the PDF export include? +
The exported PDF is a full course report: it includes a structured summary, all your Q&A, extracted code snippets, captured frames, your personal notes, smart moment markers, and timestamps — everything in one shareable document.
Do I need a credit card to start? +
No. The free plan requires no credit card. You get full access to core features with limited daily questions.
