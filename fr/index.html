Pourquoi utilise-t-on useEffect ici plutôt que useMemo ?
✦
À 12:34, le formateur utilise useEffect parce que ce code exécute un effet de bord (appel API) — pas une valeur dérivée. useMemo est destiné aux calculs qui retournent des valeurs mémoïsées. Puisque la récupération de données déclenche une action externe plutôt que de retourner un résultat calculé, useEffect est le hook approprié ici.

FAQ

Questions fréquentes

Qu'est-ce qu'AskMyTuto exactement ? +
AskMyTuto est un assistant d'apprentissage IA pour les tutoriels YouTube. Il suit votre timestamp vidéo en temps réel et vous permet de poser des questions sur le moment exact que vous regardez — vous donnant des réponses instantanées et contextuelles sans jamais quitter la page.
Fonctionne-t-il avec n'importe quelle vidéo YouTube ? +
Oui. Collez n'importe quelle URL YouTube publique et AskMyTuto récupère instantanément la transcription complète. Le support d'autres sources (fichiers vidéo, documentation) arrive bientôt.
Puis-je obtenir des réponses dans ma propre langue ? +
Oui. Vous pouvez définir votre langue de sortie préférée et chatter dedans — même si le tutoriel original est dans une langue complètement différente. AskMyTuto traduit le contexte, pas seulement les mots.
Que contient l'export PDF ? +
Le PDF exporté est un rapport de cours complet : il comprend un résumé structuré, tous vos Q&R, des extraits de code, des images capturées, vos notes personnelles, des marqueurs de moments intelligents et des timestamps — tout dans un document partageable.
Ai-je besoin d'une carte bancaire pour commencer ? +
Non. Le plan gratuit ne nécessite aucune carte bancaire. Vous obtenez un accès complet aux fonctionnalités de base avec un nombre limité de questions quotidiennes.
